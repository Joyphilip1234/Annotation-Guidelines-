# Annotation Guidelines

This repository contains sample guidelines and instructions for image annotation projects used in AI and computer vision.

## Contents

- `README.md` – Overview of annotation process.
- `sample_images/` – Images with annotation examples.
- `instructions/` – Step-by-step annotation instructions for bounding boxes, polygons, and keypoints.

## Purpose

The repository demonstrates:

- Ability to follow and document annotation guidelines.
- Standard practices in labeling images for AI datasets.
- Attention to detail and consistency in annotation tasks.

## Tools Used

- CVAT
- Labelbox
- SuperAnnotate
- Microsoft Excel (for tracking annotations)

## Notes

- Guidelines are adaptable to any computer vision project.
- This is designed for showcasing professional annotation workflow skills.
